# ProyectoRetiro

Sistema de gestión de retiro para **Villa Marista**.

## Ejecutar

```bash
npm start
```

Abrir en `http://localhost:3000`.

## Login inicial (admin único)

- Usuario: `admin`
- Contraseña: `admin123`

> ⚠️ La contraseña por defecto es solo para entorno local/desarrollo. En producción configurá `ADMIN_PASS`.

Podés cambiarlos con variables de entorno:

- `ADMIN_USER`
- `ADMIN_PASS`
